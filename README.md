# CSCI046-Final-Project
A music player that uses a custom Queue for upcoming songs and a Stack for playback history, built to demonstrate core data structures in a real-world application :)

DATA STRUCTURES USED

1) Queue — manages upcoming songs in FIFO order
2) Stack — tracks playback history in LIFO order
Both are implemented from scratch, using lecture notes / labs for reference
